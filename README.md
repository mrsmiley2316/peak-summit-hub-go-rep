# PEAK Summit Mastery Toolkit
**Open-source offline planning suite for custom mountain adventures.**

---

## Introduction

**PEAK Summit Mastery Toolkit** is a standalone collection of **offline tools** for planning routes, managing resources, and testing strategies in **custom, self-hosted environments**.

- Runs **entirely offline**  
- **No game process attachment**  
- **No input automation**  
- **No network activity**  
- MIT-licensed, fully open on GitHub  

> *"Your personal summit notebook — plan once, climb smart."* — Custom mountain builders

---

## Core Features (All Offline & Non-Invasive)

| Feature | Description |
|--------|-------------|
| **Route Planner** | Interactive path builder with drag-to-edit ledges, caches, and safe zones. Export/share configs. |
| **Resource Calculator** | Track stamina, weight, and recovery based on gear and terrain. Adjustable sliders. |
| **Biome Reference** | Offline database of terrain types, spawn rules, and environmental effects. |
| **Config Library** | 95+ preset loadouts: solo, squad, speed, survival. Import/export JSON. |
| **Local Simulator** | Run virtual climbs with bots to test paths and item placement. No server needed. |
| **Theme Pack** | 22+ visual skins: Frost, Jungle, Night Glow, etc. Apply to planner UI. |
| **Debug Console** | View logs, graphs, and simulation stats during planning. |

*All tools operate independently of any live session.*
